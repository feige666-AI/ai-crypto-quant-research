# 回测假设与指标定义

## 信号与成交时点

短期和长期均线都只使用当前及之前的收盘价。信号在当前数据行收盘时形成，并在同一收盘价基础上模拟成交。这是便于教学的简化模型，可能产生同 bar 时点偏差。

## 模拟成交价格

- 买入：`close × (1 + slippage_rate)`
- 卖出：`close × (1 - slippage_rate)`

滑点只用于成交价；持仓中的每日权益按原始收盘价估值。

## 手续费

买入和卖出都按成交金额乘以 `fee_rate` 收费。全仓买入时，持仓数量会预留买入手续费，保证现金不会因为费用变成负数。

## 未平仓与期末规则

默认 `--close-position-at-end` 为开启状态。最后仍有持仓时，使用最后收盘价、卖出滑点和手续费强制退出，交易原因写为 `end_of_data`。

关闭该选项时：

- 按最后收盘价计算最终权益。
- `open_position=true`。
- 未退出持仓不计入完成交易、胜率或 Profit Factor。

## 指标

- **最大回撤**：每行权益相对到当时为止历史峰值的跌幅，其中最大值为最大回撤。
- **基准收益率**：第一根收盘价买入并持有到最后一根收盘价的简单收益率，不扣费用。
- **市场暴露率**：回测结束每一行计算后，有持仓的行数除以总行数。
- **胜率**：净利润大于 0 的完成交易数除以全部完成交易数。没有完成交易时为 `null`。
- **Profit Factor**：正净利润之和除以负净利润绝对值之和。没有完成交易或没有亏损交易时为 `null`。

## 已知偏差

- 没有下一根开盘价成交、订单延迟和盘口深度。
- 没有部分成交、流动性约束、税费或资金费率。
- 合成数据的阶段结构是人为设计的，不是市场样本。
- 少量交易下的胜率和 Profit Factor 不稳定。

因此这些结果只能用于验证编程和研究流程，不能用于真实投资决策。
