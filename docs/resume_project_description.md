# 简历项目描述

项目名称：
AI Crypto Quant Research｜Codex辅助加密货币量化研究工具

项目链接：
https://github.com/xuanfeiw0/ai-crypto-quant-research

项目描述：
这是一个本地运行的加密货币量化研究学习 Demo，用于展示从 CSV 行情数据读取、策略信号生成到回测结果输出的基础流程。项目不连接真实交易账户，不提供投资建议，重点展示 AI 工具辅助开发、数据处理意识和项目文档整理能力。

主要工作：
1. 使用 Codex 辅助拆解项目结构，整理为适合 GitHub 展示的作品集仓库。
2. 设计本地 CSV 样例数据，避免上传真实账户、真实交易记录或隐私信息。
3. 实现基础移动平均线信号生成和本地回测流程，输出资金曲线和结果摘要。
4. 编写 README、项目状态、技术决策、任务记录和架构说明，提升项目可读性。
5. 梳理“已完成 / 计划中”功能边界，避免夸大项目能力或收益表现。

项目成果：
1. 形成一个可运行、可阅读、可上传 GitHub 的学习型项目 Demo。
2. 展示了使用 AI 工具辅助开发、整理文档和推进项目的能力。
3. 建立了基础的数据处理和结果分析流程。
