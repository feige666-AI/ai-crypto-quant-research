# PROJECT_STATE

更新时间：2026-05-13

## 当前进度

项目已整理为一个独立的 GitHub 展示型仓库，当前版本是“本地 CSV + 简单均线策略 + 基础回测输出”的学习 Demo。

## 已完成模块

- 项目目录结构：已按 README、docs、src、data、results、examples 等方式整理。
- 数据读取：已支持读取本地 OHLCV CSV，并检查必要字段。
- 策略信号：已支持短期 / 长期移动平均线交叉信号。
- 回测流程：已支持简单多头回测，输出资金曲线和摘要指标。
- 示例数据：已提供 `data/sample_data.csv`。
- 示例结果：已准备输出到 `results/backtest_result_sample.csv` 和 `results/backtest_summary_sample.json`。
- 文档：已补充 README、架构说明、决策记录、任务记录和简历文案。

## 未完成模块

- 暂未接入真实交易所 API。
- 暂未支持真实账户、实盘交易或自动下单。
- 暂未加入复杂策略、参数优化或多币种组合回测。
- 暂未加入图表可视化。
- 暂未加入完整单元测试。
- 暂未加入 Typer CLI；当前先用 Python 标准库 `argparse` 保持依赖简单。

## 当前问题

- 当前样例数据较小，只适合演示流程，不适合得出策略有效性结论。
- 当前回测逻辑非常简化，没有考虑滑点、盘口深度、资金费率等真实交易因素。
- 当前项目目录是新整理出的展示仓库，原始工作区中没有发现可直接迁移的旧量化源码。

## 上传 GitHub 前建议

- 检查 GitHub 用户名和仓库地址是否正确。
- 如需公开展示，请只上传本目录，不要上传父目录中的翻译包、PPT 项目或其他无关文件。
- 上传前再检查一次是否误放了 `.env`、私有数据、数据库文件或真实账户信息。
