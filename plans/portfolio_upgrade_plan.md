# 作品集升级执行计划

日期：2026-07-16

## 目标

把现有最小 Demo 升级为可安装、可测试、可复现、可解释的 Python 量化研究学习项目，同时保持单资产、只做多、合成数据和完全离线的安全边界。

## 执行步骤

1. 记录 `main` 分支修改前基线和已知问题。
2. 建立标准 `src` 包安装与 `argparse` CLI。
3. 完善 OHLCV 校验、均线信号、教学型回测、导出和可视化。
4. 增加可复现合成数据、示例结果、自动化测试和 GitHub Actions。
5. 重写 README 与作品集文档，并同步项目状态档案。
6. 运行安装、CLI、pytest、覆盖率、Ruff、编译和隐私扫描。
7. 通过 GitHub 工作分支提交并创建指向 `main` 的 Draft PR。

## 明确不做

- 不连接交易所，不处理 API Key，不执行真实订单。
- 不加入做空、杠杆、永续合约、多资产组合或参数寻优。
- 不加入数据库、Web 前端、云部署或机器学习预测。
- 不通过调参美化合成数据上的回测结果。

## 验收原则

- 文档命令必须实际运行。
- 不可计算指标使用 `null`，不使用 0 冒充。
- 失败结果如实记录，不能写成已完成。
- 示例数据和图表必须明确标注为 synthetic data。
